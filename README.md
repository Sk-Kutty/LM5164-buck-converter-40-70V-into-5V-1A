# High Voltage Buck Converter | LM5164 (40V–70V to 5V/1A)

## Overview

This repository contains the complete hardware design files for a **High Voltage DC-DC Buck Converter** based on the **Texas Instruments LM5164** switching regulator.

The converter is designed to accept a **40V–70V DC input** and provide a **regulated 5V DC output** capable of delivering up to **1A** continuous load current.

The project demonstrates practical skills in **power electronics, analog hardware design, PCB layout, component selection, and hardware validation**.

## Hardware Design Considerations

* Input decoupling capacitor placed close to VIN.
* Bootstrap capacitor positioned near the IC.
* Minimized switching loop area.
* Wide copper traces used for high-current paths.
* Solid ground plane to reduce noise and improve thermal performance.
* Feedback trace routed away from switching nodes.
